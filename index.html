Speecher

Speecher is a small piece of JavaScript for creating simple in-browser presentations.

Click with the mouse or press right arrow to continue..

  • Simple, lightweight
  • Built with jQuery
  • Easy to use
  • Custom styling, steps
More on the next slide - click to continue.

Simple to use:

<div class="slide">
  <h1>This is the first slide!</h1>
  It has some awesome text..
</div>

<div class="slide">
  <h1>And here's another slide</h1>
  <ul>
    <li class="step">Using steps ..</li>
    <li class="step">.. is easy</li>
  </ul>
</div>
Make sure to include the latest jQuery, jQuery UI and speecher.js.
